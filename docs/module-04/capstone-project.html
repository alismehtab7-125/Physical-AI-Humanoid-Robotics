Capstone Project: Autonomous Humanoid Agent

Project Overview​

The Capstone Project represents the culmination of the Physical AI & Humanoid Robotics course, integrating all concepts learned across the four modules. Students will develop a fully autonomous humanoid agent capable of understanding natural language commands, navigating complex environments, and executing sophisticated manipulation tasks using a combination of ROS 2, simulation environments (Gazebo/Unity), and AI planning systems (LLM/VLM).

Project Objectives​

By completing this capstone project, students will demonstrate mastery of:

  1. ROS 2 Integration: Implementing distributed robotic systems with proper communication patterns
  2. Simulation-to-Reality Transfer: Developing and testing in simulation before deployment
  3. AI Planning: Using LLMs and VLMs for high-level task understanding and planning
  4. Perception Systems: Implementing multimodal perception for environment understanding
  5. Control Systems: Developing stable humanoid locomotion and manipulation controllers
  6. Safety and Reliability: Implementing proper safety protocols and error handling

Project Scope​

The autonomous humanoid agent should be capable of:

  • Understanding natural language commands (e.g., "Go to the kitchen and bring me the red cup")
  • Navigating to specified locations while avoiding obstacles
  • Identifying and manipulating objects in the environment
  • Handling unexpected situations and recovering from failures
  • Providing feedback on task execution status

# capstone_planning/llm_task_parser.py
import rclpy
from rclpy.node import Node
from std_msgs.msg import String
from capstone_interfaces.srv import ParseCommand
from capstone_interfaces.msg import TaskPlan
import openai
import json

class LLMTaskParser(Node):
    def __init__(self):
        super().__init__('llm_task_parser')

        # Service for parsing commands
        self.parse_service = self.create_service(
            ParseCommand, 'parse_task_command', self.parse_command_callback
        )

        # Publisher for task plans
        self.plan_pub = self.create_publisher(TaskPlan, 'task_plan', 10)

        # World state subscriber
        self.world_state_sub = self.create_subscription(
            String, 'world_state', self.world_state_callback, 10
        )

        self.world_state = {}
        self.llm_client = openai.OpenAI(api_key=self.get_parameter('openai_api_key', ''))
        self.model = self.get_parameter('llm_model', 'gpt-3.5-turbo')

    def parse_command_callback(self, request, response):
        """
        Parse natural language command into structured task plan
        """
        try:
            # Get current world state
            world_state = self.world_state

            # Create LLM prompt
            prompt = self.create_parsing_prompt(request.command, world_state)

            # Call LLM
            completion = self.llm_client.chat.completions.create(
                model=self.model,
                messages=[{"role": "user", "content": prompt}],
                temperature=0.3
            )

            # Parse response
            structured_response = json.loads(completion.choices[0].message.content)

            # Create task plan
            task_plan = self.create_task_plan(structured_response)

            # Publish plan
            self.plan_pub.publish(task_plan)

            response.success = True
            response.plan = task_plan
            response.message = "Successfully parsed command"

        except Exception as e:
            response.success = False
            response.message = f"Failed to parse command: {str(e)}"

        return response

    def create_parsing_prompt(self, command, world_state):
        """
        Create prompt for LLM to parse command
        """
        return f"""
        Parse the following natural language command into a structured task plan:

        Command: "{command}"

        Current World State: {json.dumps(world_state, indent=2)}

        Provide a detailed task plan with the following structure:
        {{
            "tasks": [
                {{
                    "action": "action_type",
                    "parameters": {{"param1": "value1", ...}},
                    "description": "Human-readable description"
                }}
            ],
            "dependencies": [...],
            "success_criteria": [...]
        }}

        Action types: navigate, locate_object, grasp_object, place_object, speak, wait
        """

    def create_task_plan(self, structured_response):
        """
        Create TaskPlan message from structured response
        """
        plan_msg = TaskPlan()
        plan_msg.header.stamp = self.get_clock().now().to_msg()
        plan_msg.header.frame_id = "map"

        for task in structured_response.get("tasks", []):
            task_msg = TaskPlan.Task()
            task_msg.action = task["action"]
            task_msg.parameters = json.dumps(task.get("parameters", {}))
            task_msg.description = task.get("description", "")
            plan_msg.tasks.append(task_msg)

        return plan_msg

    def world_state_callback(self, msg):
        """
        Update world state from message
        """
        try:
            self.world_state = json.loads(msg.data)
        except json.JSONDecodeError:
            self.get_logger().warn("Failed to parse world state message")

# capstone_perception/vlm_perceptor.py
import rclpy
from rclpy.node import Node
from sensor_msgs.msg import Image
from std_msgs.msg import String
from capstone_interfaces.msg import ObjectDetection
import cv2
from cv2 import aruco
import numpy as np
import torch
from transformers import CLIPProcessor, CLIPModel

class VLMPerceptor(Node):
    def __init__(self):
        super().__init__('vlm_perceptor')

        # Initialize CLIP model
        self.clip_model = CLIPModel.from_pretrained("openai/clip-vit-base-patch32")
        self.clip_processor = CLIPProcessor.from_pretrained("openai/clip-vit-base-patch32")

        # Image subscriber
        self.image_sub = self.create_subscription(
            Image, 'camera/image_raw', self.image_callback, 10
        )

        # Object detection publisher
        self.detection_pub = self.create_publisher(ObjectDetection, 'object_detections', 10)

        # World state publisher
        self.world_state_pub = self.create_publisher(String, 'world_state', 10)

        # Object classes to detect
        self.object_classes = [
            "cup", "bottle", "book", "phone", "chair", "table",
            "kitchen", "living room", "bedroom", "refrigerator"
        ]

    def image_callback(self, msg):
        """
        Process image and detect objects using VLM
        """
        try:
            # Convert ROS image to OpenCV format
            cv_image = self.ros_image_to_cv2(msg)

            # Detect objects using CLIP
            detections = self.detect_objects_with_clip(cv_image)

            # Publish detections
            detection_msg = ObjectDetection()
            detection_msg.header = msg.header
            detection_msg.objects = detections
            self.detection_pub.publish(detection_msg)

            # Update world state
            world_state = self.update_world_state(detections, cv_image.shape)
            state_msg = String()
            state_msg.data = json.dumps(world_state)
            self.world_state_pub.publish(state_msg)

        except Exception as e:
            self.get_logger().error(f"Error processing image: {e}")

    def detect_objects_with_clip(self, image):
        """
        Use CLIP to detect objects in image
        """
        # Prepare text descriptions
        text_descriptions = [f"a photo of a {obj}" for obj in self.object_classes]

        # Process image and text
        inputs = self.clip_processor(images=image, text=text_descriptions, return_tensors="pt", padding=True)

        with torch.no_grad():
            outputs = self.clip_model(**inputs)

        # Get similarity scores
        logits_per_image = outputs.logits_per_image
        probs = logits_per_image.softmax(dim=-1).cpu().numpy()[0]

        # Create detections
        detections = []
        for i, (obj_class, prob) in enumerate(zip(self.object_classes, probs)):
            if prob > 0.1:  # Threshold
                detection = ObjectDetection.Detection()
                detection.name = obj_class
                detection.confidence = float(prob)
                detection.x = 0.0  # To be implemented with actual detection coordinates
                detection.y = 0.0
                detection.z = 0.0
                detections.append(detection)

        return detections

    def update_world_state(self, detections, image_shape):
        """
        Update world state based on detections
        """
        world_state = {
            "timestamp": self.get_clock().now().nanoseconds,
            "objects": {},
            "robot": {
                "location": "unknown",
                "battery_level": 100.0
            },
            "environment": {
                "resolution": image_shape[:2]
            }
        }

        for detection in detections:
            world_state["objects"][detection.name] = {
                "confidence": detection.confidence,
                "location": {"x": detection.x, "y": detection.y, "z": detection.z}
            }

        return world_state

# capstone_navigation/humanoid_navigator.py
import rclpy
from rclpy.node import Node
from geometry_msgs.msg import PoseStamped, Twist
from nav_msgs.msg import Path
from sensor_msgs.msg import LaserScan, PointCloud2
from capstone_interfaces.msg import TaskPlan
from visualization_msgs.msg import MarkerArray
import numpy as np
from scipy.spatial.transform import Rotation as R

class HumanoidNavigator(Node):
    def __init__(self):
        super().__init__('humanoid_navigator')

        # Navigation state
        self.current_pose = None
        self.current_plan = None
        self.plan_index = 0

        # Publishers and subscribers
        self.cmd_vel_pub = self.create_publisher(Twist, 'cmd_vel', 10)
        self.path_pub = self.create_publisher(Path, 'current_path', 10)
        self.marker_pub = self.create_publisher(MarkerArray, 'navigation_markers', 10)

        # Task plan subscriber
        self.plan_sub = self.create_subscription(
            TaskPlan, 'task_plan', self.plan_callback, 10
        )

        # Sensor subscribers
        self.laser_sub = self.create_subscription(
            LaserScan, 'scan', self.laser_callback, 10
        )

        self.odom_sub = self.create_subscription(
            PoseStamped, 'odom', self.odom_callback, 10
        )

        # Navigation parameters
        self.linear_vel = 0.3  # m/s
        self.angular_vel = 0.5  # rad/s
        self.waypoint_tolerance = 0.3  # meters
        self.rotation_tolerance = 0.1  # radians

        # Timer for navigation control
        self.nav_timer = self.create_timer(0.1, self.navigation_control)

    def plan_callback(self, msg):
        """
        Receive task plan and extract navigation tasks
        """
        for task in msg.tasks:
            if task.action == "navigate":
                params = json.loads(task.parameters)
                target_location = params.get("location", "unknown")

                # Convert location to coordinates (this would use a location map)
                target_pose = self.location_to_pose(target_location)

                if target_pose:
                    self.follow_path_to_pose(target_pose)

    def follow_path_to_pose(self, target_pose):
        """
        Generate and follow path to target pose
        """
        # For now, create a simple path to target
        current_pose = self.current_pose

        if current_pose is None:
            self.get_logger().warn("Current pose not available")
            return

        # Simple straight-line path (in a real system, this would use a proper planner)
        path = self.generate_simple_path(current_pose, target_pose)

        self.current_plan = path
        self.plan_index = 0

        # Publish path for visualization
        path_msg = Path()
        path_msg.header.stamp = self.get_clock().now().to_msg()
        path_msg.header.frame_id = "map"

        for pose in path:
            stamped_pose = PoseStamped()
            stamped_pose.header = path_msg.header
            stamped_pose.pose = pose
            path_msg.poses.append(stamped_pose)

        self.path_pub.publish(path_msg)

    def navigation_control(self):
        """
        Main navigation control loop
        """
        if self.current_plan is None or self.plan_index >= len(self.current_plan):
            # No active plan, stop the robot
            cmd_vel = Twist()
            self.cmd_vel_pub.publish(cmd_vel)
            return

        target_pose = self.current_plan[self.plan_index].pose
        current_pose = self.current_pose

        if current_pose is None:
            return

        # Calculate distance to target waypoint
        dx = target_pose.position.x - current_pose.position.x
        dy = target_pose.position.y - current_pose.position.y
        distance = np.sqrt(dx**2 + dy**2)

        # Check if reached current waypoint
        if distance < self.waypoint_tolerance:
            self.plan_index += 1
            if self.plan_index >= len(self.current_plan):
                # Path completed
                cmd_vel = Twist()
                self.cmd_vel_pub.publish(cmd_vel)
                self.get_logger().info("Navigation task completed")
                return

        # Calculate control commands
        cmd_vel = Twist()

        # Linear velocity towards target
        if distance > self.waypoint_tolerance:
            cmd_vel.linear.x = min(self.linear_vel, distance * 0.5)

        # Angular velocity for heading correction
        target_heading = np.arctan2(dy, dx)
        current_heading = self.quaternion_to_yaw(current_pose.orientation)
        heading_error = self.normalize_angle(target_heading - current_heading)

        cmd_vel.angular.z = max(-self.angular_vel, min(self.angular_vel, heading_error * 2.0))

        # Publish command
        self.cmd_vel_pub.publish(cmd_vel)

    def laser_callback(self, msg):
        """
        Handle laser scan data for obstacle avoidance
        """
        # Implement obstacle detection and avoidance
        min_distance = min(msg.ranges)

        if min_distance < 0.5:  # Obstacle too close
            # Emergency stop or path replanning
            cmd_vel = Twist()
            cmd_vel.linear.x = 0.0
            cmd_vel.angular.z = 0.5  # Try to rotate away
            self.cmd_vel_pub.publish(cmd_vel)

    def odom_callback(self, msg):
        """
        Update current pose from odometry
        """
        self.current_pose = msg.pose

    def quaternion_to_yaw(self, quaternion):
        """
        Convert quaternion to yaw angle
        """
        q = [quaternion.w, quaternion.x, quaternion.y, quaternion.z]
        r = R.from_quat([q[1], q[2], q[3], q[0]])  # [x, y, z, w] format
        return r.as_euler('xyz')[2]  # yaw is the z component

    def normalize_angle(self, angle):
        """
        Normalize angle to [-pi, pi] range
        """
        while angle > np.pi:
            angle -= 2 * np.pi
        while angle < -np.pi:
            angle += 2 * np.pi
        return angle

    def location_to_pose(self, location_name):
        """
        Convert location name to coordinates (simplified)
        """
        location_map = {
            "kitchen": (5.0, 2.0, 0.0),
            "living_room": (0.0, 0.0, 0.0),
            "bedroom": (-3.0, 4.0, 0.0),
            "office": (2.0, -3.0, 0.0)
        }

        if location_name in location_map:
            x, y, theta = location_map[location_name]
            pose = Pose()
            pose.position.x = x
            pose.position.y = y
            pose.position.z = 0.0

            # Convert theta to quaternion
            from tf_transformations import quaternion_from_euler
            quat = quaternion_from_euler(0, 0, theta)
            pose.orientation.w = quat[0]
            pose.orientation.x = quat[1]
            pose.orientation.y = quat[2]
            pose.orientation.z = quat[3]

            return pose

        return None

# capstone_manipulation/manipulator.py
import rclpy
from rclpy.node import Node
from std_msgs.msg import String
from geometry_msgs.msg import Pose, Point
from capstone_interfaces.msg import ObjectDetection, TaskPlan
from sensor_msgs.msg import JointState
import numpy as np

class Manipulator(Node):
    def __init__(self):
        super().__init__('manipulator')

        # Publishers and subscribers
        self.joint_cmd_pub = self.create_publisher(JointState, 'joint_commands', 10)
        self.task_sub = self.create_subscription(TaskPlan, 'task_plan', self.task_callback, 10)
        self.detection_sub = self.create_subscription(ObjectDetection, 'object_detections', self.detection_callback, 10)

        # Current object detections
        self.current_detections = []

        # Robot kinematic parameters (simplified)
        self.link_lengths = [0.3, 0.3, 0.2]  # Link lengths for simple arm model

    def task_callback(self, msg):
        """
        Handle manipulation tasks from plan
        """
        for task in msg.tasks:
            if task.action in ["grasp_object", "place_object"]:
                params = json.loads(task.parameters)

                if task.action == "grasp_object":
                    self.grasp_object(params["object_name"])
                elif task.action == "place_object":
                    self.place_object(params["location"])

    def detection_callback(self, msg):
        """
        Update object detections
        """
        self.current_detections = msg.objects

    def grasp_object(self, object_name):
        """
        Execute grasping motion for specified object
        """
        # Find object in current detections
        target_object = None
        for obj in self.current_detections:
            if obj.name == object_name:
                target_object = obj
                break

        if target_object is None:
            self.get_logger().warn(f"Object {object_name} not found")
            return

        # Calculate grasp pose
        grasp_pose = self.calculate_grasp_pose(target_object)

        # Execute reaching motion
        self.move_to_pose(grasp_pose)

        # Close gripper
        self.close_gripper()

        self.get_logger().info(f"Successfully grasped {object_name}")

    def place_object(self, location):
        """
        Place currently held object at specified location
        """
        # Calculate placement pose based on location
        place_pose = self.calculate_place_pose(location)

        # Execute placement motion
        self.move_to_pose(place_pose)

        # Open gripper
        self.open_gripper()

        self.get_logger().info(f"Successfully placed object at {location}")

    def calculate_grasp_pose(self, object_detection):
        """
        Calculate appropriate grasp pose for object
        """
        # For a cup, approach from above with gripper aligned
        grasp_pose = Pose()
        grasp_pose.position.x = object_detection.x
        grasp_pose.position.y = object_detection.y
        grasp_pose.position.z = object_detection.z + 0.1  # Approach from above

        # Orientation: gripper facing down
        grasp_pose.orientation.w = 1.0  # No rotation
        grasp_pose.orientation.x = 0.0
        grasp_pose.orientation.y = 0.0
        grasp_pose.orientation.z = 0.0

        return grasp_pose

    def calculate_place_pose(self, location):
        """
        Calculate placement pose for location
        """
        # Simplified: place at location coordinates at table height
        place_pose = Pose()

        # Convert location to coordinates
        location_coords = self.location_to_coordinates(location)
        place_pose.position.x = location_coords[0]
        place_pose.position.y = location_coords[1]
        place_pose.position.z = 0.8  # Table height

        # Same orientation as grasp
        place_pose.orientation.w = 1.0
        place_pose.orientation.x = 0.0
        place_pose.orientation.y = 0.0
        place_pose.orientation.z = 0.0

        return place_pose

    def move_to_pose(self, target_pose):
        """
        Move manipulator to target pose (simplified)
        """
        # In a real system, this would use inverse kinematics
        # For simulation, we'll just publish joint angles
        joint_state = JointState()
        joint_state.name = ['joint1', 'joint2', 'joint3', 'gripper']
        joint_state.position = [0.0, 0.0, 0.0, 0.0]  # Placeholder values

        self.joint_cmd_pub.publish(joint_state)

    def close_gripper(self):
        """
        Close the gripper
        """
        joint_state = JointState()
        joint_state.name = ['gripper']
        joint_state.position = [0.01]  # Closed position
        self.joint_cmd_pub.publish(joint_state)

    def open_gripper(self):
        """
        Open the gripper
        """
        joint_state = JointState()
        joint_state.name = ['gripper']
        joint_state.position = [0.05]  # Open position
        self.joint_cmd_pub.publish(joint_state)

    def location_to_coordinates(self, location):
        """
        Convert location name to coordinates
        """
        location_map = {
            "kitchen_counter": (4.5, 1.5),
            "kitchen_table": (5.5, 2.5),
            "living_room_table": (0.5, 0.5),
            "bedroom_nightstand": (-2.5, 3.5)
        }

        return location_map.get(location, (0.0, 0.0))

# capstone_simulation/sim_bridge.py
import rclpy
from rclpy.node import Node
from std_msgs.msg import String
from sensor_msgs.msg import Image, LaserScan, JointState
from geometry_msgs.msg import Twist, PoseStamped
from capstone_interfaces.msg import ObjectDetection
import numpy as np

class SimulationBridge(Node):
    def __init__(self):
        super().__init__('simulation_bridge')

        # Publishers for simulation data
        self.image_pub = self.create_publisher(Image, 'camera/image_raw', 10)
        self.laser_pub = self.create_publisher(LaserScan, 'scan', 10)
        self.joint_state_pub = self.create_publisher(JointState, 'joint_states', 10)
        self.odom_pub = self.create_publisher(PoseStamped, 'odom', 10)

        # Subscribers for commands
        self.cmd_vel_sub = self.create_subscription(Twist, 'cmd_vel', self.cmd_vel_callback, 10)
        self.joint_cmd_sub = self.create_subscription(JointState, 'joint_commands', self.joint_cmd_callback, 10)

        # Simulation state
        self.robot_pose = np.array([0.0, 0.0, 0.0])  # x, y, theta
        self.joint_positions = np.zeros(19)  # Humanoid joint positions

        # Timer for simulation updates
        self.sim_timer = self.create_timer(0.01, self.simulation_step)  # 100 Hz

    def simulation_step(self):
        """
        Main simulation update step
        """
        # Update robot pose based on current velocity commands
        self.update_robot_dynamics()

        # Publish sensor data
        self.publish_camera_data()
        self.publish_laser_data()
        self.publish_joint_states()
        self.publish_odometry()

    def update_robot_dynamics(self):
        """
        Update robot dynamics based on commands
        """
        # Simplified dynamics model
        dt = 0.01  # Time step

        # Update pose based on velocity (simplified)
        if hasattr(self, 'current_cmd_vel'):
            vx = self.current_cmd_vel.linear.x
            wz = self.current_cmd_vel.angular.z

            # Update position
            self.robot_pose[0] += vx * np.cos(self.robot_pose[2]) * dt
            self.robot_pose[1] += vx * np.sin(self.robot_pose[2]) * dt
            self.robot_pose[2] += wz * dt

    def publish_camera_data(self):
        """
        Publish simulated camera data
        """
        # In a real simulation, this would come from the graphics engine
        # For this example, we'll publish empty data
        img_msg = Image()
        img_msg.header.stamp = self.get_clock().now().to_msg()
        img_msg.header.frame_id = 'camera_frame'
        img_msg.height = 480
        img_msg.width = 640
        img_msg.encoding = 'rgb8'
        img_msg.step = 640 * 3  # Width * bytes per pixel
        img_msg.data = [0] * (480 * 640 * 3)  # Empty image data

        self.image_pub.publish(img_msg)

    def publish_laser_data(self):
        """
        Publish simulated laser scan data
        """
        scan_msg = LaserScan()
        scan_msg.header.stamp = self.get_clock().now().to_msg()
        scan_msg.header.frame_id = 'laser_frame'
        scan_msg.angle_min = -np.pi / 2
        scan_msg.angle_max = np.pi / 2
        scan_msg.angle_increment = np.pi / 180  # 1 degree
        scan_msg.time_increment = 0.0
        scan_msg.scan_time = 0.1
        scan_msg.range_min = 0.1
        scan_msg.range_max = 10.0

        # Generate simulated ranges (simplified)
        num_ranges = int((scan_msg.angle_max - scan_msg.angle_min) / scan_msg.angle_increment) + 1
        scan_msg.ranges = [5.0] * num_ranges  # All ranges at 5m (no obstacles)

        self.laser_pub.publish(scan_msg)

    def publish_joint_states(self):
        """
        Publish joint state information
        """
        joint_state = JointState()
        joint_state.header.stamp = self.get_clock().now().to_msg()
        joint_state.name = [f'joint_{i}' for i in range(len(self.joint_positions))]
        joint_state.position = self.joint_positions.tolist()

        self.joint_state_pub.publish(joint_state)

    def publish_odometry(self):
        """
        Publish odometry information
        """
        odom_msg = PoseStamped()
        odom_msg.header.stamp = self.get_clock().now().to_msg()
        odom_msg.header.frame_id = 'map'
        odom_msg.pose.position.x = float(self.robot_pose[0])
        odom_msg.pose.position.y = float(self.robot_pose[1])
        odom_msg.pose.position.z = 0.0

        # Convert theta to quaternion
        from tf_transformations import quaternion_from_euler
        quat = quaternion_from_euler(0, 0, self.robot_pose[2])
        odom_msg.pose.orientation.w = quat[0]
        odom_msg.pose.orientation.x = quat[1]
        odom_msg.pose.orientation.y = quat[2]
        odom_msg.pose.orientation.z = quat[3]

        self.odom_pub.publish(odom_msg)

    def cmd_vel_callback(self, msg):
        """
        Handle velocity commands
        """
        self.current_cmd_vel = msg

    def joint_cmd_callback(self, msg):
        """
        Handle joint commands
        """
        # Update joint positions based on commands
        for i, name in enumerate(msg.name):
            try:
                idx = self.joint_positions.index(name)  # This is simplified
                self.joint_positions[idx] = msg.position[i]
            except ValueError:
                # Joint name not found in our list
                pass

Evaluation Criteria​

Technical Requirements (70%)​

  • ROS 2 Integration (15%): Proper use of ROS 2 concepts, message passing, and architecture
  • Perception System (15%): Effective use of VLM for object detection and scene understanding
  • Planning System (15%): LLM-guided task planning and decomposition
  • Navigation (10%): Reliable navigation in complex environments
  • Manipulation (10%): Successful object grasping and manipulation
  • Simulation Integration (5%): Proper simulation-to-reality considerations

Performance Metrics (20%)​

  • Task Success Rate: Percentage of successfully completed tasks
  • Execution Time: Efficiency of task completion
  • Robustness: Ability to handle unexpected situations
  • Safety: Proper safety protocols and emergency procedures

Documentation and Presentation (10%)​

  • Code Quality: Well-documented, maintainable code
  • System Documentation: Clear architecture and design documentation
  • Final Presentation: Comprehensive demonstration of capabilities
